## 📌 Overview

The **Smart & Integrated Hostel Management System** is a web-based application designed to automate and streamline hostel operations. It provides an efficient platform for managing student accommodation, room allocation, complaints, fees, and administrative tasks.

This system reduces manual work, improves data accuracy, and enhances communication between students and hostel authorities.

---

## 🚀 Features

* 🔐 **User Authentication**

  * Secure login and registration for students, warden, and admins

* 🛏️ **Room Management**

  * Allocate and manage rooms
  * Track room availability in real-time

* 👨‍🎓 **Student Management**

  * Store and manage student records
  * Maintain hostel history

* 📅 **Attendance Management**

  * Track student attendance efficiently

* 🛠️ **Complaint Management**

  * Students can raise complaints
  * Admin can view and resolve issues

* 💳 **Fee Management**

  * Track fee payments and pending dues

* 🔔 **Notifications**

  * Announcements and updates for students

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/aaditi72/Smart-and-Integrated-Hostel-Management-System.git
```

### 2️⃣ Move to project folder

```
cd Smart-and-Integrated-Hostel-Management-System
```

### 3️⃣ Setup Environment

* Ensure you have the required runtime installed (Node.js or PHP)
* Configure database connection in project files

### 4️⃣ Import Database

* Use MySQL Workbench or any MySQL client
* Create a database (e.g., `hostel_db`)

### 5️⃣ Run Project

* Start backend server
  
* Open browser:

```
http://127.0.0.1:5000
```
